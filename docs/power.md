# 供电（强烈建议先看）

舵机“抽动 / 乱转 / 复位 / 慢慢漂移”最常见的原因不是程序，而是供电与地线回流。

## 推荐供电方案

- BlackPill：用板子自己的 USB 供电（或稳定的 3.3V LDO）
- PCA9685 逻辑：`VCC = 3.3V`（跟 STM32 一致）
- 舵机电源：`V+ = 5~6V` 独立电源（建议 >= 3A，6 个舵机更建议更大余量）

## 共地与走线

- 必须共地：舵机电源 GND ↔ PCA9685 GND ↔ STM32 GND
- 建议星形接地：让舵机大电流回路尽量不要经过 STM32 的地线/调试线

## 电容与抗干扰建议

在 PCA9685 的 `V+` 与 `GND` 端附近并：
- 1000uF~2200uF 电解（耐压 >= 10V）
- 可再并一个 0.1uF 陶瓷

## 关于 ST-Link 供电（重要坑）

不建议用 ST-Link 的 `3.3V` 给板子供电（电流能力有限，容易不稳）。

推荐：
- BlackPill 用自己的 USB 供电
- ST-Link 只接：`SWDIO`、`SWCLK`、`GND`（可选 `NRST`）
